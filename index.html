    Traffic Capture Infrastructure

    Traffic Capture Infrastructure (TCI) is a system for creating and processing datasets from network traffic across a set of distributed nodes.

    Get started

    Created distributed datasets

    Use TCI to create datasets of network traffic across a wide variety of distributed devices.

    Process captured data

    Process the captured data using your predefined scripts in a sandboxed Docker environment.

    Highly configurable

    The TCI system can be controlled from your web browser, or from libraries in Python, Go, or Rust.
